
# 一、斜率与导数

### 1.1 直线斜率

对于直线而言，斜率是固定的几何量，计算公式为
$m=\frac{y_2-y_1}{x_2-x_1}$ 表示纵坐标变化量与横坐标变化量的比值‌此时斜率与导数无关，仅描述直线的倾斜程度。

### 1.2 曲线切线斜率

对于函数图像上的曲线，某点处的导数值等于该点切线的斜率。导数的定义是函数在某点的瞬时变化率，其几何意义即为切线斜率‌
例如，若函数
$f(x)$ 在
$X_0$处可导则 
$f'\left(x_0\right)$即为该点切线斜率‌

斜率可以用来描述函数曲线在不同点上的变化速率。在微积分中，斜率被称为导数，可以通过导数来计算函数在某一点的变化率，并用于解决最优化、极值、速度、加速度等问题。

- 直线斜率是静态几何量，与导数无关；
- 曲线某点的导数值动态反映该点切线斜率，二者在几何意义上是等价的‌



# 二、求导

## 2.1 x²求导

导数的定义是： 
$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$ 

对于$f(x) = x^2$，代入： 
$f'(x) = \lim_{h \to 0} \frac{(x+h)^2 - x^2}{h}$


**1.展开
$(x+h)^2$:**

*  $(x+h)^2 = x^2 + 2xh + h^2$


**2.代入分子：**

*  $f'(x) = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h}$


**3.简化分子取消
$x^2$：**

*  $f'(x) = \lim_{h \to 0} \frac{2xh + h^2}{h}$



4.因式分解h： 
*  $f'(x) = \lim_{h \to 0} \frac{h(2x + h)}{h}$



**5.取消h（因为
$h \neq 0$）：**

*  $f'(x) = \lim_{h \to 0} (2x + h)$



**6.取极限（当 
$h \to 0$）：**

$f'(x) = 2x$

## 2.2 复合函数求导

复合函数就像“函数的函数”，比如 y = sin(x²)，它是由 y = sin(u) 和 u = x² 这两个函数复合而成。

如果变量 y 是 u 的函数，而 u 又是 x 的函数，即： y = f(u) 和 u = g(x)，则复合函数 y = f(g(x)) 的导数为：

$\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$

或者写成：
$f(g(x))' = f'(g(x)) \cdot g'(x)$

**外层的导数 × 内层的导数**

### 示例1：y = sin(x²)求导
1.  **识别内外函数**
    *   **外层函数：** `f(u) = sin(u)` (u是中间变量)
    *   **内层函数：** `u = g(x) = x²`

2.  **分别求导**
    *   **外层导数：** `df/du = cos(u)` (对 `sin(u)` 求导)
    *   **内层导数：** `du/dx = 2x` (对 `x²` 求导)

3.  **应用链式法则（相乘）**
    *   `dy/dx = (df/du) * (du/dx) = cos(u) * 2x`

4.  **回代（用内层函数替换u）**
    *   `dy/dx = cos(x²) * 2x = 2x cos(x²)`

**所以，`(sin(x²))' = 2x cos(x²)`**

### 示例2：带幂次的复合函数 y = (3x + 1)¹⁰
1.  **识别内外函数**
    *   **外层函数：** `f(u) = u¹⁰`
    *   **内层函数：** `u = g(x) = 3x + 1`

2.  **分别求导**
    *   **外层导数：** `df/du = 10u⁹` (幂法则)
    *   **内层导数：** `du/dx = 3` (对 `3x+1` 求导)

3.  **应用链式法则**
    *   `dy/dx = 10u⁹ * 3 = 30u⁹`

4.  **回代**
    *   `dy/dx = 30(3x + 1)⁹`

**所以，`((3x + 1)¹⁰)' = 30(3x + 1)⁹`**


### 示例3：多层复合函数 `y = e^(sin x)`

这个函数可以看作两层复合。

1.  **识别内外函数**
    *   **最外层：** `f(v) = e^v`
    *   **中间层：** `v = sin(u)`
    *   **最内层：** `u = x`

2.  **分别求导**
    *   `df/dv = e^v` (指数函数求导不变)
    *   `dv/du = cos(u)` (对 `sin(u)` 求导)
    *   `du/dx = 1` (对 `x` 求导)

3.  **应用链式法则（层层相乘）**
    *   `dy/dx = (df/dv) * (dv/du) * (du/dx) = e^v * cos(u) * 1`

4.  **回代** (先换成 `u`，再换成 `x`)
    *   先把 `v` 换回 `sin(u)`： `dy/dx = e^(sin u) * cos(u)`
    *   再把 `u` 换回 `x`： `dy/dx = e^(sin x) * cos(x)`

**所以，`(e^(sin x))' = e^(sin x) * cos(x)`**


# 三、偏导

偏导数就是研究‌多个变量‌(多元函数)同时存在时，函数随‌其中一个变量‌的变化率，而其他变量保持不变‌

## 求偏导的步骤与示例

**基本规则**：在求关于某个变量的偏导数时，将所有其他变量视为常数。

### 示例1：基础多项式

`f(x, y) = x² + 3xy + y³`

#### 求对 x 的偏导数 $f_x$
- 把 `y` 当作常数
- `x²` 的导数是 `2x`
- `3xy` 中的 `3y` 是常数系数，导数是 `3y`
- `y³` 是常数，导数是 `0`
- **结果**：
  $f_x = \frac{\partial f}{\partial x} = 2x + 3y$

#### 求对 y 的偏导数 $f_y$
- 把 `x` 当作常数
- `x²` 是常数，导数为 `0`
- `3xy` 中的 `3x` 是常数系数，导数是 `3x`
- `y³` 的导数是 `3y²`
- **结果**：
  $f_y = \frac{\partial f}{\partial y} = 3x + 3y²$

### 示例2：指数与三角函数

`f(x, y) = e^x * sin(y)`

#### 求 $f_x$
- 把 `sin(y)` 当作常数
- `e^x` 的导数是 `e^x`
- **结果**：
  $f_x = e^x * sin(y)$

#### 求 $f_y$
- 把 `e^x` 当作常数
- `sin(y)` 的导数是 `cos(y)$
- **结果**：
  $f_y = e^x * cos(y)$

### 示例3：商法则的应用

`f(x, y) = x / (x² + y²)`

使用商的求导法则：
$(u/v)' = (u'v - uv') / v²$

#### 求 $f_x$
- 令 `u = x`, `v = x² + y²`
- `u_x = 1` （对x求偏导，y是常数）
- `v_x = 2x` （对x求偏导，y是常数）
- **结果**：
  $f_x = \frac{(1)(x²+y²) - (x)(2x)}{(x²+y²)²} = \frac{y² - x²}{(x²+y²)²}$

#### 求 $f_y$
- `u = x`, `v = x² + y²`
- `u_y = 0` （对y求偏导，x是常数）
- `v_y = 2y` （对y求偏导，x是常数）
- **结果**：
  $f_y = \frac{(0)(x²+y²) - (x)(2y)}{(x²+y²)²} = \frac{-2xy}{(x²+y²)²}$


## 高阶偏导数
和一元函数一样，我们可以对偏导数再次求偏导，得到高阶偏导数。对于 `z = f(x, y)`：
### 二阶偏导
- 对x求两次：$\frac{\partial}{\partial x}(\frac{\partial f}{\partial x}) = \frac{\partial² f}{\partial x²} = f_{xx}$
- 对y求两次：$\frac{\partial}{\partial y}(\frac{\partial f}{\partial y}) = \frac{\partial² f}{\partial y²} = f_{yy}$
- **混合偏导**：先对x求，再对y求：$\frac{\partial}{\partial y}(\frac{\partial f}{\partial x}) = \frac{\partial² f}{\partial y \partial x} = f_{xy}$
- **混合偏导**：先对y求，再对x求：$\frac{\partial}{\partial x}(\frac{\partial f}{\partial y}) = \frac{\partial² f}{\partial x \partial y} = f_{yx}$
